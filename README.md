# TailoredCV

TailoredCV is a web application designed to help users streamline their job search process by generating tailored cover letters and resumes. Using the power of OpenAI's GPT API, TailoredCV takes a user’s uploaded resume and a job description to produce a customized cover letter and updated resume in seconds.

---

## Features

- **Upload Resume**: Users can upload their existing resume in PDF or DOCX format.
- **Paste Job Description**: Enter the job description directly into the application.
- **AI-Powered Customization**: Leverages OpenAI’s GPT API to tailor your resume and cover letter to the specific job.
- **Download Results**: Easily download the tailored cover letter and resume for your application.
